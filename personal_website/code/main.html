Hydranet: AI for Fingernail Condition Detection

Hydranet is a deep learning model designed to analyze fingernail images and classify potential conditions. It utilizes convolutional, regression, and classification networks to learn subtle visual features like hue, streak intensity, and streak ratios of the nail.

The project was my first major ML architecture, and uses a variety of problem-solving techniques to inspect valuable information from something so seemingly simple, as a fingernail.
